wait
========================

.. code-block:: text

	wait(sec)


Arguments
------------

* sec
	* The number of seconds to wait for.

Description
-------------

Waits for the provided number of seconds.

Sample
-------------

.. code-block:: json

	{
        "row": 1,
        "command": "wait",
        "args": [
        	"1.5"
        ],
        "end": 1
    }
